# Deep PR Codex Pack

Этот пакет расширяет faststart-репозиторий и добавляет 100 пронумерованных PR-задач для работы с Codex.

## Как использовать
1. Всегда давай Codex только один текущий zip.
2. Проси сделать только один PR за раз.
3. После ответа Codex приноси новый zip обратно в ChatGPT.
4. Я проверю прогресс и скажу следующий номер.

## Что уже есть
- PR-001..PR-028 — core route для fastest working repo.
- PR-029..PR-100 — глубокие расширения по семействам.

## Самое важное
Начинай с PR-001.
Не пропускай ранние foundation PR без причины.
