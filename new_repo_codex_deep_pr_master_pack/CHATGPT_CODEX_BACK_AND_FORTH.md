# ChatGPT <-> Codex workflow

- ChatGPT:
  - определяет следующий PR
  - проверяет zip от Codex
  - готовит следующий handoff
- Codex:
  - делает только один PR
  - возвращает новый zip
- Пользователь:
  - передаёт zip туда и обратно

## Правило
Long PR data держать внутри zip, не в длинном сообщении.
