# Timezones (UTC) and monotonic time audit

- Все времена в данных и логах храним в UTC (`*_utc`).
- Любая конвертация в локальное время — только на уровне UI/BI.

Canonical chain:
`signal_time_utc → local_send_time_utc → first_confirm_time_utc`

Если нарушено:
- `failure_mode=time_skew`
- счётчик метрики + алерт
